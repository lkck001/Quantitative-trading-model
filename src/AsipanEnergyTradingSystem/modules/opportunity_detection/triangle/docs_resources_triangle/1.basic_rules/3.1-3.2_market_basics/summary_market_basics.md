# 3.1–3.2 Market Basics (Energy Perspective)

## 1. Core Concepts (定义)
- **市场波动成因**：价格波动由多空力量对比造成；交易本质是“跟随市场最强的一方”。
- **无向论**：市场在任意时刻同时存在三种方向可能（上涨、下跌、横向运行）。
- **趋势三分**：行情分为上升趋势、下降趋势、无趋势三类，并在三者间交替转换。
- **均线状态**：
  - 上升趋势 → 均线系统向上发散排列
  - 下降趋势 → 均线系统向下发散排列
  - 无趋势 → 均线系统收敛
- **能量论**：市场运行在“能量积累”与“能量释放”两种状态之间交替转换，这是市场运行的基本规律。

## 2. 判定规则（可执行）
- **趋势识别**
  - IF 多条均线收敛 → THEN 进入无趋势状态（对应能量积累）。
  - IF 均线系统向上发散 → THEN 上升趋势（对应向上释放）。
  - IF 均线系统向下发散 → THEN 下降趋势（对应向下释放）。
- **能量积累判定**（满足多数特征即可判定）
  - 价格在固定区间内波动、振幅小、运行时间长。
  - 运行角度接近水平（≈0°）。
  - 内部结构复杂、上下反复、K线阴阳交错且密集。
  - 心理特征：行情沉闷压抑，多空都难以盈利。

## 3. 关键参数（可量化）
- **积累时间占比**：积累阶段占整个行情运行时间的 **≥80%**（相对释放而言）。
- **振幅阈值**：价格波动幅度小于某个“区间阈值”（建议后续用 ATR/百分比定义）。
- **角度阈值**：积累区间整体角度 **接近 0°**。
- **K线密度**：阴阳交错、K线密集（可用单位时间内K线实体/影线重叠度衡量）。

## 4. 输出字段（用于识别模块）
- `state_type`: accumulation / release / unknown
- `direction`: up / down / sideways
- `range_upper`, `range_lower`
- `duration_bars`, `duration_ratio`
- `slope/angle`
- `volatility_metric`
- `confidence`

## 5. 在识别流程中的位置
- **前置过滤层**：先判断市场处于积累还是释放。
- **结构识别层**：积累区间的“边界线 + 触及点”作为后续标准机会识别基础。

## 7. 典型案例要点（书中示例提炼）
- **螺纹钢积累案例**：
  - A/B 两段积累区间；区间内波动小、触及上下边线即折返。
  - 区间角度接近 0°，内部结构复杂。
  - 积累时间明显长于前一段快速行情；脱离区间后出现快速行情。
- **橡胶积累案例**：
  - 上方出现“假突破”：价格上破后迅速拉回并快速下跌（逆释放）。
  - B 区域为“无结构能量积累”：区间狭小、K线极度密集、几乎无内部结构。
- **铁矿石积累案例**：
  - 两段积累，结构清晰；在区间上边线附近形成清晰触及点。
  - 区间近水平，积累时间 > 前一段快速行情的 **2 倍**。
  - 为标准交易机会提供结构基础。
